Mairard
Character Registration Bot
Initial Setup
I
Invite the Bot
Add Mairard to your Discord server via the OAuth invite link.
II
/setup #channel
Set where all logs appear — registrations and admin actions both go here.
III
Ready
No roles or channels to create. Players register inside Ticket Tool tickets.
Ticket Flow
1
Open Ticket
Player opens a ticket via Ticket Tool.
›
2
Add Characters
Player runs /addchar for each character.
/addchar
›
3
Auto Log
Each addition is posted to the log channel.
›
4
Approval
Admin reviews and assigns LCF or OCF role manually.
›
5
Done
Ticket gets closed by Ticket Tool.
Player Commands
/addchar [cat] [name] [class] [clan]
LOGGED Register a character. Pick LCF or OCF, then fill in the details.
/remove_char
LOGGED Pick a character from your list to remove.
/my_chars
See all your registered characters, split by LCF and OCF.
/help
Show all available commands and the ticket flow guide.
Admin Commands
/setup [channel]
Set the log channel for all bot activity.
/admin_add [@user] [cat] ...
LOGGED Add a character for any player. Assigns LCF or OCF role automatically.
/admin_delete [@user] [cat]
LOGGED Select a player's character to soft-delete.
/admin_edit [@user] [cat]
LOGGED Edit a player's character via a pre-filled form.
/list
Full character list split by LCF and OCF.
/search [query]
Search by character name, clan or Discord user. Leave empty for the full list.
/deleted_chars
See the last 50 removed characters with their removal dates.
/export
Download all active registrations as a CSV file.
/import_data [file]
LOGGED Bulk-import registrations by uploading a CSV.
/purge
DESTRUCTIVE Wipe all registration data for this server. Requires confirmation.
Notes
Soft DeletesRemoved characters stay in the database. Full history available via /deleted_chars.
Unique NamesCharacter names are unique per player, case-insensitive.
Role Assignment/admin_add assigns roles automatically. The /addchar flow is manual by design.
Guild LockedBot auto-leaves any server it's not authorised for. All data is server-isolated.
